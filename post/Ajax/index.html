readystate

0 open（）方法还未被调用

1 send（）方法还未被调用

2 send（）已经被调用，响应头和响应状态已经返回，

3 正在下载响应体

4 请求已经把响应下载完毕

Json and Js. 区别

Json是一种数据格式化的语言

  1. Json没有抄袭js 的function 和undefine
  2. JSON的字符串首尾必须是双引号 “”
  3. 不能表示复杂的对象，只能表示Hash
  4. 没有原型链

​ 返回的字符串，字符串的格式刚好符合 JSON 的规范

同源策略

当向baidu。com发送AJax请求时。如果不是一样的域名，就会被拒绝，

可得 ： 只有 协议+端口+域名 一摸一样才允许法AJAX请求

浏览器 的网络都必须支持同源策略，以此来保证服务器和客户端的私密信息的安全
